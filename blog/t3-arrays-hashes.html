POST 3 - [array] vs. {hash}

Saturday, June 6, 2015

An array is a way to itemize objects with integers in an orderly fashion (i.e. a key to a value). A hash is a way to itemize two objects together (i.e. a value to another value, or a product to information about that product) without having it to be just an integer value.

So for all the archivists out there, a hash would be a more viable option when setting up a system for your archived items - a more refined way to itemize objects with values beyond integers. Just saying..
